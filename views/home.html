How to use Tinker.

Simply enter a name of your chose and a room id. To chat with someone make sure that all of you enter the same room id correctly. Also Tinker will not save any chat activity and because of Hypertext Transfer Protocol Secure your messages will be encrypted. Note that this is just a demo version
